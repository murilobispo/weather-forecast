## About The Project

This is a **Minimalist Weather Dashboard** built with Streamlit, designed to provide real-time meteorological data through a clean and intuitive user experience. The application accepts **city-based inputs** to display temperature, rain probability, and wind data.

[![Open in Streamlit](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://murilobispo-weather-forecast.streamlit.app)

## Why Streamlit?
I chose **Streamlit** for its data-optimized components and seamless integration with **Pandas**. This combination allows for powerful data manipulation and a reactive frontend using pure Python, prioritizing speed and a minimalist workflow.

## Data Sources
* **[Open-Meteo](https://open-meteo.com/):** High-resolution weather forecast data (No API key required).
* **[Nominatim (OpenStreetMap)](https://nominatim.org/):** Open-source geocoding used to lookup city coordinates.

## Installation
1. Clone the repo
    ```sh
    git clone https://github.com/murilobispo/weather-forecast.git
    ```
    
2. Create and activate a virtual environment (Recommended)
    ```sh
    python -m venv venv
    # Linux / Mac
    source venv/bin/activate
    # Windows
    venv\Scripts\activate
    ```
    
3. Install dependencies
    ```sh
    pip install -r requirements.txt
    ```
    
4. Run the app
    ```sh
    streamlit run app.py
    ```
